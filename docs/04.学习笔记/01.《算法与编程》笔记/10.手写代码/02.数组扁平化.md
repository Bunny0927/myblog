###### 1. ES6 flat
返回一个新数组,包含数组与提取嵌套数组的所有元素的新数组。参数使用`Infinity`，可展开任意深度的嵌套数组。
> ```js
> array.flat(Infinity)
> ```

###### 2. 循环递归（面试写法）
设置depth参数用于控制递归的深度
> ```js
> function flatten(arr, depth) {
> let result = []
> arr.forEach((item, i, arr) => {
>   if(Array.isArray(item) && depth > 0) {
>     result = result.concat(flatten(item, depth - 1))
>   } else {
>     result.push(arr[i])
>   }
> });
> return result;
> }
> ```
